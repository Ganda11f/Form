Perguntas
  1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?

    2. R= o atributo <placeholder> serve para digitar algo dentro do campo da tag <input>.


  2. Na tag <input> com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

    3. R= Ele serve para pré-selecionar uma caixa quando a página é carregada.


  3. Na tag <input> para que serve o atributo "value"?

    4. R= Ele serve para colocar um valor, deixar algo escrito na barra quando a página é carregada.


  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. R= "Name"


  5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

    6. R= A tag <label> serve para escrever algo no formulário. o atributo "for" serve para especificar em qual caixa o texto ficará colado.


  6. Na tag <option> para que serve o atributo "selected"?

    7. R= O atributo "selected" serve para criar uma lista de "drop-down" (caixa de escolha no formulario).


  7. Nas tags de formulário para que serve o atributo "required"?

    8. R= O atributo ""required" serve para especificar que o usuário deve preencher o campo com o atributo para enviar.


  8. Na tag <form> para que serve o atributo "action"?

    9. R= O atributo especifica para onde enviar os dados do formulário quando um formulário é enviado.


  9. Na tag <form> para que serve o atributo "method"?

    10. R= O atributo especifica como enviar dados de formulário, pode ser enviado com as variáveis "get" ou "post".


  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    11. R= "password".


  11. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?

    12. R= Os dados do formulário podem ser enviados como variáveis ​​de URL (com method="get") ou como transação de postagem HTTP (com method="post").
